# 💰 Expense Tracker

Приложение для отслеживания личных расходов с графическим интерфейсом на Tkinter.

## 👤 Автор
**Ткаченко Дмитрий** 

## 📋 Описание

Expense Tracker — это десктопное приложение, которое позволяет:
- Добавлять расходы с указанием суммы, категории, даты и описания
- Просматривать все добавленные расходы в таблице
- Фильтровать расходы по категории и диапазону дат
- Подсчитывать общую сумму расходов и сумму за выбранный период
- Сохранять данные между запусками программы

## 🚀 Установка и запуск

### Требования
- Python 3.6 или выше
- Tkinter (входит в стандартную библиотеку Python)

### Запуск

```bash
# Клонирование репозитория
git clone https://github.com/ваш_логин/expense-tracker.git
cd expense-tracker

# Запуск программы
python main.py
